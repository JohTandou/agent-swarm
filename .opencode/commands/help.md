Explains the Agent Swarm pipeline to the user:

**5 available routes:**
- DIRECT: you respond directly (questions, explanations)
- SIMPLE: 1-2 files, no contract (crons, scripts, isolated fix)
- ADAPT: copy/adapt from another project
- MEDIUM: single-domain feature (front only or back only)
- FULL: complete feature front + back + DB

**Available agents:**
- search: codebase analysis + framework docs (Context7)
- planner: planning + architectural choices
- contract: TypeScript types + OpenAPI spec + SQL migrations
- front: UI implementation (21st.dev Magic + ui-ux-pro-max)
- back: backend implementation + scripts/crons
- tester: test execution + error categorization
- reviewer: security + quality gate before commit
- writer: documentation (CHANGELOG, API.md, ARCHITECTURE.md)
- build: agent orchestration

**Available commands:**
/status — recent run metrics
/docs — update documentation now
/help — this help
