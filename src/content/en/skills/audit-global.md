## Role

The global audit performs an exhaustive review of the application across six critical dimensions: implementation completeness (what is delivered vs what is planned), production readiness, security, performance, code quality and marketing/SEO. It produces a consolidated report with actionable recommendations.

## Use Cases

- **Pre-launch checklist**: verify that an application is ready for production deployment
- **Technical due diligence**: evaluate an existing project before takeover or investment
- **Continuous improvement**: identify priority improvement areas
- **Cross-cutting code review**: combine code quality, performance and security in a single pass

## Triggers

The planner agent loads this skill when:
- The user requests a complete assessment of their application
- A global audit or "health check" request is made

## Inputs

- Complete codebase (frontend, backend, configuration, CI/CD)
- Specification files and documentation
- Production metrics if available

## Outputs

- Multi-dimensional audit report with per-axis scores
- Prioritized list of fixes (critical, major, minor)
- Strategic recommendations for production deployment
