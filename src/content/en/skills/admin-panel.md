## Role

The admin-panel skill analyzes the existing codebase (backend, database, API, authentication) and generates a complete admin panel. It includes CRUD operations on all entities, analytical dashboards, user management and role-based access control (RBAC). The result is a production-ready dashboard, adapted to the project architecture.

## Use Cases

- **New project**: scaffold a complete admin panel in one command
- **Existing project**: add an administration interface to an already deployed API
- **Migration**: replace a legacy admin panel with a modern version
- **Rapid prototyping**: get a functional management interface in minutes

## Triggers

The planner agent loads this skill when:
- The user requests admin panel generation
- A backend management interface task is assigned

## Inputs

- Existing codebase (backend, database schema, API)
- Authentication and roles configuration
- Entity specifications to administer

## Outputs

- Complete admin dashboard with CRUD for each entity
- Analytics pages with charts and metrics
- User management and RBAC access control
- Integrated, production-ready code
