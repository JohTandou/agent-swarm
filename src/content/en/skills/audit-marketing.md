## Role

The marketing and SEO audit analyzes the visibility and optimization of the application for search engines and marketing channels. It covers technical SEO (tagging, performance, indexing), marketing readiness (landing pages, conversion, analytics) and web presence optimization (social media, content, domain authority).

## Use Cases

- **Product launch**: verify that SEO is optimized before going live
- **Visibility audit**: understand why an application doesn't attract organic traffic
- **Conversion optimization**: improve landing page conversion rates
- **Content strategy**: align web content with target keywords

## Triggers

The planner agent loads this skill when:
- The user wants to analyze the SEO or marketing readiness of their application
- A web presence optimization question is asked

## Inputs

- Application URL and key pages
- Existing SEO configuration (meta tags, sitemap, robots.txt)
- Marketing analysis tools if available (Google Analytics, Search Console)

## Outputs

- Technical SEO report with prioritized fixes
- Marketing readiness analysis (landing pages, CTA, conversion funnel)
- Recommendations for web presence optimization
