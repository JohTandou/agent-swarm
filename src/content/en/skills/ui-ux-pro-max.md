## Role

UI/UX Pro Max is a design intelligence skill that provides exhaustive visual recommendations for interface creation. It covers the entire design spectrum: styles, color palettes, typographic pairings, charts, and tech stacks.

## Features

- **67 visual styles**: glassmorphism, claymorphism, minimalism, brutalism, neumorphism, bento grid, dark mode, responsive, skeuomorphism, flat design
- **96 color palettes**: a palette for every mood and every business context
- **57 typographic pairings**: validated display + body font combinations
- **25 chart types**: for dashboards and data visualization
- **13 tech stacks**: React, Next.js, Vue, Svelte, SwiftUI, React Native, Flutter, Tailwind, shadcn/ui

## Use Cases

- Plan the design of a new page or component
- Improve existing UI with style recommendations
- Choose a color palette suited to the context
- Select a premium typographic pairing
- Implement components with shadcn/ui

## Integrations

### shadcn/ui MCP
Component and example search directly from the shadcn/ui library. Usable to generate accessible and customizable components.

### Supported Action Types
- `plan`, `build`, `create`, `design`: from-scratch design
- `implement`, `review`, `fix`: work on existing code
- `improve`, `optimize`, `enhance`, `refactor`: continuous improvement
- `check`: UI/UX quality validation

## Target Project Types
- Website, landing page
- Dashboard, admin panel
- E-commerce, SaaS
- Portfolio, blog
- Mobile application

## Example

```
"I need a dark mode dashboard with glassmorphism cards"
→ UI/UX Pro Max analyzes the context
→ Proposes an adapted dark palette
→ Suggests glassmorphism components
→ Provides the corresponding Tailwind code
```
