# Swarm Wiki

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Site](https://img.shields.io/badge/site-swarm--wiki.vercel.app-C4780D)](https://swarm-wiki.vercel.app)

Documentation technique du **Swarm** — un pipeline de 9 agents IA spécialisés qui
collaborent pour concevoir, implémenter, tester et documenter des projets
logiciels de bout en bout. Ciblé recruteurs, tech leads et managers.

## Quick Start

```bash
git clone https://github.com/JohTandou/swarm-wiki.git
cd swarm-wiki
npm install
npm start
```

→ [http://localhost:3000](http://localhost:3000)

## Stack

| Domaine | Technologie |
|---|---|
| Framework | Angular 19 (standalone) |
| CSS | Tailwind v4 + SCSS |
| Animations | GSAP (ScrollTrigger) |
| Markdown | ngx-markdown + marked.js |
| Recherche | Fuse.js (fuzzy, client-side) |
| Déploiement | Vercel |
